# FundChain

**A Decentralized Crowdfunding Platform with Milestone-Based Smart Contract Governance**

FundChain is a blockchain-powered crowdfunding DApp that enhances **trust**, **transparency**, and **accountability** in fundraising. Funds are held in escrow via smart contracts and released only after verified milestones and donor approval.

Built as a final-year Computer Science project at **McPherson University**.

---

## ✨ Key Features

- **Milestone-Based Funding** – Funds are released only when predefined milestones are completed and approved
- **Smart Contract Escrow** – Donations are locked securely until conditions are met
- **Donor Voting** – Contributors can vote on milestone completion
- **IPFS Integration** – Milestone proofs and documents are stored on IPFS for permanent verification
- **Full On-Chain Transparency** – All transactions and fund flows are publicly verifiable
- **Wallet Integration** – Seamless MetaMask connection
- **Modern Frontend** – Clean and responsive user interface
